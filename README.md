# Project Overview

This project investigates customer purchasing behavior using the **Online Shoppers Purchasing Intention Dataset**. Through statistical methods and probability theory, the study aims to identify key factors influencing online purchase decisions and evaluate their significance using data-driven techniques.

The analysis combines descriptive statistics, probability distributions, hypothesis testing, regression analysis, and data visualization to uncover meaningful insights into customer conversion behavior.

## Key Findings

- **PageValues** was identified as the strongest predictor of purchase behavior.
- **ExitRates** demonstrated a strong negative relationship with purchase completion.
- New visitors showed higher conversion tendencies compared to returning visitors.
- Correlation and regression analyses revealed statistically significant relationships among several variables.
- Central Limit Theorem and hypothesis testing validated important behavioral patterns observed in the dataset.
